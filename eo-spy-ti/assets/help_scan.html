Scan View Help

Ensure that Bluetooth low energy is supported on this device and the Bluetooth Adapter is turned on. The EOSpy-TI BLE SensorTag App will start device discovery (scanning) automatically. The EOSpy-TI SensorTag Client will read all sensors from the TI-SensorTag Bluetooth LE device. EOSpy-TI will transmit GPS Position and all available TI-SenorTag sensors:

  • Temperature Sensor
  • IR Object Temp
  • Humidity Sensor
  • Barometer Sensor
  • Ambient Light Level
  • Accelerometer
  • Gyroscope
  • Magnetometer
  • SensorTag Buttons
  • Magnetometer

Scanning can be restarted any time by tapping the Scan button. During scanning Received Signal Strength Indicator (RSSI) values are continuously updated.

When a device in the list is selected, the EOSpy-TI Client application will attempt to connect to it. If the connection succeeds, the Device View is launched and the data from the enabled sensors are displayed. If connection times out, an error message is displayed. This happens when the SensorTag doesn't advertise; (Always be sure to push the side button on the SensorTag. This button toggles advertising).

The options menu settings configure the EOSpy-TI service settings. The About dialog will provide information about the SensorTag and the Android device name and Android build.
