Sensor View Help

When the view is started a Service Discovery is initiated. On Service Discovery, completion available sensor images appear and the sensor data is displayed next to them. To configure the sensors, tap the sensor to configure an on/off button and a period slider will appear. Tap the sensor again to exit the configuration menu.

The Preferences tab in the options menu is used to configure the EOSpy-TI client.

Enter the EOSpy Server address; Domain name or IP address into the Server address field. Next add this device in the EOSpy Server by entering the Device name and the Device identifier.
